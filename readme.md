### 🖥 메인 페이지 구성

#### Section 1 - 메인 비주얼  
##### 몰입감 있는 첫인상 제공
홈페이지 상단에는 연구소 전경을 담은 영상을 배경으로 사용하여 방문자의 시선을 사로잡고, 브랜드의 첫인상을 강하게 전달합니다. <br> 
영상은 자동 재생되며, 음성 없이도 분위기를 전달할 수 있도록 자연 풍경과 연구시설이 조화를 이루는 컷으로 구성하였습니다. <br> 

##### 주요 예약 유도 요소 배치
영상 하단에는 휴양림 예약 및 교육원 프로그램 신청 버튼을 시각적으로 강조된 스타일로 배치해 사용자 행동을 자연스럽게 유도합니다. <br> 
주요 기능에 대한 접근성을 높여 실질적인 이용률 향상을 도모하였습니다.<br> 

#### Section 2 - 공지사항 
##### 정보 전달 중심의 게시판 구성
공지사항 영역에서는 연구소의 최신 소식과 운영 정보를 제공하며, 방문자들이 필요한 정보를 쉽게 파악할 수 있도록 리스트형 UI를 구성하였습니다.<br> 

##### 체험학습 신청 게시판 연동
별도의 체험학습 신청 항목을 공지사항과 함께 배치하여,<br>
교육 프로그램 신청이 필요한 사용자가 자연스럽게 연결될 수 있도록 기획했습니다. 행정적인 정보 제공과 실질적인 사용자 행동을 연결하는 구조입니다.<br> 

#### Section 3 - 연구활동
##### Swiper.js 기반 슬라이드 구현
연구소에서 진행 중인 다양한 연구활동을 한눈에 확인할 수 있도록 Swiper 슬라이드를 도입하였습니다.<br> 
각 슬라이드에는 제목, 간략한 설명, 대표 이미지가 포함되어 있어 콘텐츠별로 시각적 차별화를 주었습니다.<br> 

##### 슬라이드 인터랙션 강화
마우스 스크롤 또는 터치로 콘텐츠를 넘기며 자연스럽게 탐색할 수 있으며,<br> 
자동 슬라이드 기능도 함께 제공해 정적인 정보 전달을 넘어 동적인 UI로 사용자의 관심을 유지시켰습니다.<br> 


#### Section 4 - 시설안내도 
##### 인터랙티브 맵 UI 구성
연구소 내부 지도를 기반으로 클릭 가능한 시설 안내도를 제작하여,<br> 
특정 위치 클릭 시 해당 시설의 정보가 팝업 또는 별도 영역에 표시되도록 구성하였습니다.

##### 시각적 몰입도 향상
지도를 단순한 이미지가 아닌 인터랙티브 콘텐츠로 구성함으로써<br> 
정보 전달력과 사용자의 몰입감을 동시에 높였으며, 사용자 흥미 유도에도 효과적입니다.


#### Section 5 - 갤러리  
#####식물원 기반 비주얼 콘텐츠 제공
연구소 산하 식물원에서 촬영된 실제 방문자 사진들을 슬라이드로 구성하여, <br> 
홈페이지 방문자가 방문 전 체험할 수 있는 풍경에 대한 기대감을 가질 수 있도록 설계했습니다.

##### Swiper 기반 슬라이드 갤러리
사진은 Swiper를 통해 자동 슬라이드되며, 터치 또는 클릭으로 직접 넘기며 감상할 수 있습니다.<br> 
풍경 중심의 콘텐츠로 감성적 접근을 유도하고 브랜드 이미지를 강화합니다.

#### Section 6 - 연계기관 배너
#####  슬라이드 형태의 배너 정렬
연구소와 협력 중인 연계기관의 배너를 Swiper 슬라이드 형태로 정렬하여,<br> 
한정된 공간 내에서 다양한 기관 정보를 효과적으로 노출할 수 있도록 구성하였습니다.

#####  자동 슬라이드 + 수동 탐색 기능 제공
일정 시간 간격으로 자동 슬라이드가 진행되며, 사용자 조작을 통해 앞뒤로 탐색도 가능합니다. 
<br> 배너 이미지 클릭 시 해당 기관 홈페이지로 이동되어 연계 기능도 확보하였습니다.

#### 연구소 소개 페이지  
연구소의 정체성과 철학을 효과적으로 전달하기 위해 인삿말, 연구소 연혁, 비전 등 핵심 정보를 세 개의 섹션으로 구성하였습니다.<br> 
각 항목은 방문자가 쉽게 이해할 수 있도록 간결하고 명확한 텍스트 중심으로 배치하였습니다.

#####탭 메뉴 UI 구현
콘텐츠 간의 이동을 자연스럽게 유도하기 위해 탭 메뉴를 도입하였습니다.<br>  사용자가 인삿말, 연혁, 비전 중 원하는 정보를 클릭 한 번으로 전환할 수 있도록 설계하여, 정보 접근성을 높이고 페이지 내 탐색 효율을 향상시켰습니다.
탭 전환 시 자연스러운 애니메이션 효과를 적용해 사용자 경험을 부드럽게 하였습니다.

#### 공지사항
##### 공지사항 목록 UI/UX 설계
사용자 편의성을 고려해 공지사항 목록을 깔끔하고 직관적으로 디자인하였으며, <br> 제목, 날짜, 요약 내용을 함께 배치하여 가독성을 높였습니다.

##### 페이지네이션 기능 적용
전체 공지 수가 많아질 경우를 대비해 페이지네이션을 구현하였으며, 한 페이지당 공지 10개씩 출력되도록 설정했습니다.<br>  이전/다음 버튼과 현재 페이지 표시로 사용자 경험을 향상시켰습니다.
